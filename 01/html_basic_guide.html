🌐 HTML 기본 구조 쉽게 이해하기

초등학생도 쉽게 배우는 웹사이트 만들기!

📌 HTML이 뭐예요?

HTML은 웹사이트의 뼈대예요.

레고 블록으로 집을 짓듯이, HTML로 웹사이트의 구조를 만드는 거죠.

🍱 HTML 문서 = 도시락 상자

HTML 문서를 도시락 상자라고 생각해봐요.

도시락 상자 HTML 문서
상자 겉면에 붙은 이름표 <head> (눈에 안 보이는 정보)
실제로 먹는 음식들 <body> (화면에 보이는 내용)

🏗️ 기본 구조 살펴보기

<!-- "이건 HTML5 문서예요!" 라고 알려주는 표시 -->
<!DOCTYPE html>
<html>                 <!-- HTML 문서의 시작! -->

  <head>               <!-- 머리 부분: 눈에 안 보이는 정보들 -->
    <title>내 첫 웹사이트</title>  <!-- 브라우저 탭에 보이는 제목 -->
  </head>

  <body>               <!-- 몸통 부분: 화면에 실제로 보이는 것들 -->
    <h1>안녕하세요!</h1>           <!-- 큰 제목 -->
    <p>나는 HTML을 배우는 중이에요.</p>  <!-- 문단 (글 내용) -->
  </body>

</html>                <!-- HTML 문서의 끝! -->

🔍 각 부분 쉽게 이해하기

1 <!DOCTYPE html>

"이 문서는 최신 HTML이에요!" 라고 컴퓨터에게 알려주는 거예요.

(옛날에는 복잡했는데, 지금은 이 한 줄이면 돼요!)

2 <html> ... </html>

전체 문서를 감싸는 가장 큰 상자예요.

시작 태그 <html>과 끝 태그 </html> 사이에 모든 내용을 넣어요.

3 <head> ... </head>

눈에 안 보이는 정보를 넣는 곳이에요.

  • 웹사이트 제목 (<title>)
  • 글꼴 정보
  • 검색 엔진용 설명

도시락 상자 뚜껑 안쪽에 붙은 메모 같은 거예요!

4 <body> ... </body>

화면에 실제로 보이는 모든 것을 넣는 곳이에요!

  • 글자
  • 사진
  • 버튼
  • 동영상

도시락 안의 실제 음식들이죠! 🍱

🏷️ 태그(Tag)가 뭔가요?

HTML은 태그라는 걸로 내용을 감싸요.

<태그이름>  내용  </태그이름>
  ↑시작              ↑끝

예시:

  • <h1> ~ </h1> : 큰 제목 (heading 1)
  • <p> ~ </p> : 문단 (paragraph)

💡 꼭 기억하세요! 시작 태그가 있으면 끝 태그가 필요해요!
(예외도 있지만 처음에는 다 닫아주는 게 좋아요)

🌳 한눈에 보는 전체 구조

📄 HTML 문서
│
├── 🏷️ <!DOCTYPE html> ← "이건 HTML5 문서입니다"
│
└── 📦 <html>
│
├── 🧠 <head> ← 머리 (눈에 안 보임)
│ └── <title>제목</title>
│
└── 🎨 <body> ← 몸통 (화면에 보임!)
├── <h1>제목</h1>
├── <p>글 내용</p>
└── <img>사진</img>

🎯 연습해보기!

아래 코드를 복사해서 .html 파일로 저장하고,
브라우저(크롬, 엣지 등)로 열면 웹사이트가 나타나요!

<!DOCTYPE html>
<html>
  <head>
    <title>나의 첫 홈페이지</title>
  </head>
  <body>
    <h1>🎉 환영합니다!</h1>
    <p>나는 초등학생이고, HTML을 배우고 있어요.</p>
    <p>앞으로 멋진 웹사이트를 만들 거예요!</p>
  </body>
</html>
