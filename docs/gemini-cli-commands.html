Gemini CLI Slash Commands

Gemini CLI에서 사용할 수 있는 주요 명령어 가이드

1. 세션 및 시스템 관리

CLI의 일반적인 정보 확인 및 종료를 위한 명령어입니다.

명령어설명
/aboutCLI의 버전 정보를 표시합니다.
/help (또는 /?)사용 가능한 명령어 목록과 도움말을 표시합니다.
/settings설정 편집기를 열어 설정을 수정합니다.
/stats현재 세션의 상세 통계(시간, 토큰 등)를 표시합니다.
/quit (또는 /exit)CLI를 종료합니다.
/clear터미널 화면과 세션 기록을 지웁니다.

2. 대화 기록 및 세션 제어

이전 대화로 돌아가거나 맥락을 관리합니다.

명령어설명
/resume이전 세션을 탐색하여 재개하거나 체크포인트를 관리합니다.
/rewind대화 기록을 뒤로 탐색하여 이전 상태로 되돌립니다.
/restore도구 실행 직전의 상태로 프로젝트 파일을 복구합니다.
/compress현재 대화 맥락을 요약하여 토큰을 절약합니다.
/copy마지막 출력을 클립보드에 복사합니다.

3. 모델 및 지능형 에이전트 관리

사용할 AI 모델과 서브에이전트 설정을 관리합니다.

명령어설명
/model사용할 모델을 설정하거나 구성을 관리합니다.
/agents서브에이전트 목록을 관리하고 활성화/비활성화합니다.
/skills에이전트 스킬(전문 워크플로우)을 관리합니다.
/mcpMCP 서버와 도구를 관리하고 인증합니다.

4. 작업 공간 및 컨텍스트 관리

프로젝트 구조와 AI가 참조할 메모리를 관리합니다.

명령어설명
/directory작업 공간에 디렉토리를 추가하거나 확인합니다.
/memoryAI의 계층적 메모리를 확인하거나 새로고침합니다.
/init프로젝트 전용 지침 파일(GEMINI.md) 생성을 도와줍니다.
/tools사용 가능한 도구 목록과 상세 설명을 표시합니다.

5. 기타 유틸리티

다양한 편의 기능을 제공합니다.

명령어설명
/plan계획 모드(Plan Mode)에서 생성된 계획을 확인합니다.
/themeCLI의 시각적 테마를 변경합니다.
/bug버그 리포트를 작성합니다.
/docs공식 문서를 브라우저에서 엽니다.
/vim입력창의 Vim 모드를 토글합니다.
