🎨 class와 id, 무엇이 다를까요?

초등학생도 5분 만에 이해하는 CSS 단짝 친구들!

👕 Class (클래스): 우리 반 유니폼!

클래스는 여러 명(태그)이 함께 입을 수 있는 옷과 같아요.

축구팀 친구들이 모두 같은 유니폼을 입는 것처럼, 여러 태그에 똑같은 스타일을 주고 싶을 때 사용해요.

⚽

"우리 팀은 모두 파란색 유니폼(class='blue-team')을 입자!"

.blue-team { color: blue; }

<p class="blue-team">철수</p>
<p class="blue-team">영희</p> <-- 여러 명 가능!
🆔 ID (아이디): 나만의 이름표!

아이디는 오직 한 명(태그)만 가질 수 있는 특별한 이름이에요.

우리 학교에 학생은 많지만, '3학년 1반 5번'은 딱 한 명뿐인 것과 같아요. 딱 하나만 있는 특별한 부분에 사용해요.

👑

"왕관(id='king')은 오직 한 사람만 쓸 수 있어!"

#king { font-size: 30px; }

<h1 id="king">나는 왕이다</h1> <-- 딱 한 번만 사용!
⚔️ 한 눈에 비교하기

1. 기호가 달라요! 클래스는 점(.), 아이디는 샵(#)으로 시작해요.

2. 개수가 달라요! 클래스는 여러 번, 아이디는 딱 한 번만 써야 해요.

3. 힘이 달라요! 아이디는 클래스보다 힘이 더 세서, 둘이 싸우면 아이디가 이겨요!

🎯 이것만 기억하세요!

  • Class: 여러 번 쓸 수 있는 단체복 (마침표 . 사용)
  • ID: 딱 한 번만 쓰는 이름표 (해시 # 사용)
  • 한 명의 친구가 여러 벌의 옷(class)을 껴입을 수도 있어요!
